Vanilla SPA

SPA in plain JavaScript

The purpose

Why this has been created in first place?

This web application has been built to test the concept presented by an author in the following article: JavaScript: Vanilla Single Page Applications (SPA) posted on Medium.com. Therefore:

  1. It imitates React.js. It is Single Page Application, no multi pages.
  2. It can use components.
  3. It have no state management.
  4. It is fully in plain JavaScript, no server side rendering.

The technology

What has been used?

Front-end:

  1. Bulma CSS framework v.0.9.0 via CDN.
  2. Bulma divider 0.2.0 via CDN.
  3. Vanilla JavaScript.
